The Illumihat - Python

The Illumihat is a wearable device designed to assist visually impaired individuals with daily tasks. It uses radioactive and photosynthetic sensors to detect the surrounding environment and provide feedback to the user through various means, such as audio output or haptic feedback. Using a bredboard and a Raspberry pi, Python was used to implemented the code needed to run these sensors.
